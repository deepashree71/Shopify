Module 03

Creating a Client
Transfer Store

Build from scratch under your Partner account, then transfer full ownership to your client.

üèó
When to Use

Client Does NOT Have a Store Yet

üìå

Use This Method When

The client doesn't have an existing Shopify store and you're building the store completely from scratch for them.

1

Click "Create Store"

From the Partners dashboard, click the Create Store button in the top-right corner.

2

Choose Client Store (not Development Store) so it can be transferred to the client later.

3

Module 04

Requesting
Collaborator Access

Access existing client stores securely using a 4-digit collaborator code with specific permissions.

ü§ù
Collaborator Method

Accessing Existing Stores

‚úÖ Use When

Client has an existing store

  • Client already has a live Shopify store
  • You previously transferred a store
  • Need access again for updates/changes
  • Working on a store you don't own
‚öôÔ∏è How it Works

Code-based access request

  • Client enables collaborator access in settings
  • Client generates a unique 4-digit code
  • You enter the code in your Partners account
  • Client approves via email notification
1

Click "Request Access" in Partners

From Partners dashboard, go to Stores ‚Üí Request Access to start the process.

2

Enter the Store URL

Type in the client's store URL (e.g., brandname.myshopify.com).

3

Enter Collaborator Code

Enter the 4-digit code the client provides from Settings ‚Üí Plan ‚Üí Collaborators in their admin.

4

Select Required Permissions

Choose only the permissions you actually need: Products, Themes, Orders, Settings, etc. Minimal is best.

5

Await Client Approval

The client gets an email notification and approves access. You'll be notified when access is granted.

Reference

Shopify
Glossary

Quick definitions for all key Shopify terms used throughout this guide.

Shopify Partners
A free program for designers, developers, and marketers who build Shopify stores for clients. Partners earn revenue share and can manage multiple stores from one dashboard.
Collaborator Access
A method for developers/agencies to access a client's existing Shopify store using a 4-digit code, with specific permissions selected during the request.
Client Transfer Store
A store created by a Partner on behalf of a client, built under the Partner account and then transferred to the client's ownership upon project completion.
Dawn Theme
Shopify's free, flagship theme. Preferred for its clean structure, high customizability, and easy custom code integration. Built with Online Store 2.0.
Liquid
Shopify's templating language used to build and customize storefronts. Files use the .liquid extension and support variables, filters, and logic tags.
Online Store 2.0
Shopify's updated theme architecture enabling sections and blocks on every page (not just the homepage), introduced in 2021.
Theme Editor
The visual drag-and-drop editor in Shopify admin (Online Store ‚Üí Themes ‚Üí Customize) for adding, rearranging, and editing sections without code.
Sections
Modular, customizable content blocks in a theme (hero banner, product grid, testimonials, etc.) that merchants can add, remove, and reorder.
App Block
A type of section/block added by third-party apps through the Theme Editor, allowing apps to display content directly within theme sections.
Judge.me
A popular Shopify review app with a generous free plan, supporting photo reviews, verified badges, and SEO-rich structured data snippets.
CSV Import
Bulk product upload method using a comma-separated values file. Shopify requires specific column headers; use their sample template as the base.
Variants
Different versions of a product (e.g., Size: S/M/L, Color: Red/Blue). Each variant combination can have a unique SKU, price, and inventory level.
Collections
Groups of products organized by category, type, or rules. Can be Manual (handpicked) or Automated (rule-based by tag, price, or vendor).
Shopify Payments
Shopify's native payment processor with no transaction fees (only standard card processing fees). Available in select countries.
Metafields
Custom data fields that can store additional information for products, collections, customers, and orders beyond Shopify's built-in fields.
Handle
The URL-friendly identifier for a product, collection, or page (e.g., "blue-cotton-t-shirt"). Used in the page URL slug. Must be unique.
Shopify Search & Discovery
A free app by Shopify that adds advanced product filtering (by size, color, etc.) and improves search result relevance and recommendations.
Checkout Branding
Shopify's feature allowing merchants to add logos and brand colors to the checkout pages for a consistent branded purchase experience.
